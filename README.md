# 🚀 Windows Auto-Deployment & Post-Install Manager

Этот репозиторий содержит комплексное решение для полностью автоматической установки и настройки Windows 10. Проект базируется на файле ответов (`autounattend.xml`) для автоматизации этапа инсталляции ОС и кастомном PowerShell-загрузчике с графическим интерфейсом (GUI), который берет на себя всю рутину по настройке системы сразу после первого запуска.

Цель проекта — принцип **Zero-touch deployment**: настроил один раз, запустил и на выходе получил полностью готовую к работе, оптимизированную машину со всем необходимым софтом.

---

## 💿 Быстрый старт (Установка Windows)

Для развертывания вам понадобится любой «чистый» (оригинальный) образ Windows 10. Вы можете скачать его с официального сайта Microsoft или проверенных ресурсов вроде [comss.ru](https://www.comss.ru/).

1. Запишите образ Windows на флешку (например, через Rufus).
2. Возьмите файл **`autounattend.xml`** (находится прямо в корне этого репозитория) и просто скопируйте его в **корень вашей загрузочной флешки**.
3. Загрузитесь с флешки. 

> ⚠️ **ВАЖНО: Безопасность данных**
> Весь процесс установки Windows и создания профилей полностью автоматизирован, **за одним исключением — выбор и разметка диска оставлены ручными.** Это сделано специально для вашей безопасности, чтобы автоматика случайно не отформатировала разделы с вашими личными файлами или резервными копиями.

## 🛸 Главный Загрузчик (GUI Диспетчер)

Все рабочие PowerShell-скрипты, которые система использует для настройки, находятся в папке `Scripts` в этом репозитории. За их выполнение отвечает **Главный Загрузчик** — стильный менеджер автоматизации с графическим интерфейсом.

**Функции Загрузчика:**
*   **Оркестрация:** Строгий последовательный запуск дочерних скриптов.
*   **Визуализация:** Отображение красивого нелагающего GUI с анимированным прогресс-баром и статусом каждой задачи (без страшных черных консолей).
*   **Логирование:** Незаметная фоновая запись всех действий в лог-файл для отладки.
*   **Отказоустойчивость:** Перехват ошибок и продолжение работы даже если один из модулей дал сбой.

💡 **Универсальность:** Загрузчик **можно использовать как самостоятельный скрипт**, не переустанавливая Windows! Если у вас уже стоит чистая ОС, вы можете просто скачать папку `Scripts` к себе на ПК, запустить менеджер, и он автоматически выполнит всю оптимизацию и установит нужный софт.

## ⚙️ Архитектура модулей (Дочерние скрипты)

После загрузки рабочего стола Загрузчик по очереди вызывает следующие модули из папки `Scripts`:

*   🧹 **`clean-and-photo.ps1` — Оптимизация ОС**
    Жесткая блокировка телеметрии Windows, удаление встроенного мусора (Debloat), применение оптимальных настроек реестра и возвращение классического средства просмотра фотографий.
    
*   🧩 **`install-sys-components.ps1` — Системные компоненты**
    Тихая установка критически важных сред выполнения, библиотек и зависимостей, необходимых для стабильной работы игр и профессионального софта.

*   📦 **`apps-install.ps1` — Базовый пакет ПО**
    Надежная "тихая" (silent) установка повседневных программ. Автоматически скачивает по прямым ссылкам и разворачивает: *Google Chrome, Steam (интегрируется, но отключается в автозагрузке), WinRAR, qBittorrent, ShareX, K-Lite Codec Pack и RMS Host*.
    
*   📝 **`office-install.ps1` — Развертывание Microsoft Office**
    Скрипт для полностью автоматической загрузки, установки и локальной активации пакета Microsoft Office 2024 Professional Plus LTSC. 

*   🔥 **`reset-setup-scripts.ps1` — Финальная зачистка**
    Скрипт "уборки". Удаляет временные инсталляторы, очищает кэш и **полностью удаляет саму папку `Scripts`** с диска, оставляя систему в идеальном порядке перед финальной перезагрузкой ПК.

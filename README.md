# 🚀 Windows Auto-Deployment & Post-Install Manager

Этот репозиторий содержит комплексное решение для полностью автоматической установки и настройки Windows 10. Проект базируется на использовании файла ответов (`unattend.xml`) для автоматизации этапа установки ОС и кастомном PowerShell-менеджере с графическим интерфейсом (GUI), который берет на себя всю рутину по настройке системы сразу после первого запуска.

Цель проекта — принцип **Zero-touch deployment**: настроил один раз, запустил и на выходе получил полностью готовую к работе, оптимизированную машину со всем необходимым софтом.

## 🖥️ Особенности
*   **Стильный GUI-диспетчер:** Написанный на чистом PowerShell менеджер автоматизации с современным темным интерфейсом, плавной анимацией прогресс-бара и защитой от случайного закрытия.
*   **Глубокая автоматизация:** От установки ОС через `unattend.xml` до скачивания и тихой распаковки программ без единого диалогового окна.
*   **Отказоустойчивость:** Встроенная проверка целостности файлов (SHA256), система повторных попыток скачивания при обрывах связи и подробное фоновое логирование процессов.

## ⚙️ Архитектура модулей (Дочерние скрипты)

Процесс пост-установочной настройки логически разделен на несколько независимых `.ps1` файлов. Главный диспетчер запускает их строго по очереди:

*   🧹 **`clean-and-photo.ps1` — Оптимизация ОС**
    Жесткая блокировка телеметрии Windows, удаление встроенного мусора (Debloat), применение оптимальных настроек реестра и возвращение классического средства просмотра фотографий.
    
*   🧩 **`install-sys-components.ps1` — Системные компоненты**
    Тихая установка критически важных сред выполнения, библиотек и зависимостей, необходимых для стабильной работы игр и профессионального софта.

*   📦 **`apps-install.ps1` — Базовый пакет ПО**
    Модуль надежной "тихой" (silent) установки повседневных программ. Автоматически скачивает по прямым ссылкам и разворачивает: *Google Chrome, Steam (интегрируется, но отключается в автозагрузке), WinRAR, qBittorrent, ShareX, K-Lite Codec Pack и RMS Host*.
    
*   📝 **`office-install.ps1` — Развертывание Microsoft Office**
    Скрипт для полностью автоматической загрузки, установки и локальной активации пакета Microsoft Office 2024 Professional Plus LTSC. 

*   🔥 **`reset-setup-scripts.ps1` — Финальная зачистка**
    Скрипт "уборки". Удаляет временные инсталляторы, очищает кэш и полностью удаляет саму папку с установочными скриптами, оставляя систему в идеальном порядке перед финальной перезагрузкой.

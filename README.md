# 🚀 Windows Auto-Deployment & Post-Install Manager

Этот репозиторий представляет собой комплексное решение для полностью автоматической установки, настройки и оптимизации Windows 10. Проект объединяет этап развертывания ОС с помощью файла ответов (`autounattend.xml`) и постобработку системы через кастомный интерактивный PowerShell-диспетчер с анимированным GUI.

**Главная цель проекта:** один запуск — и на выходе вы получаете идеально чистую, стабильную, оптимизированную систему со всеми необходимыми библиотеками, базовым софтом и офисным пакетом, полностью готовую к работе.

---

## ⚡ Способы запуска (Универсальность)

Систему автоматизации можно инициализировать двумя удобными путями в зависимости от вашей задачи:

### Способ 1. Установка Windows «с нуля»

Для развертывания вам понадобится любой «чистый» (оригинальный) образ Windows 10. Вы можете скачать его с официального сайта Microsoft или проверенных ресурсов вроде [comss.ru](https://www.comss.ru/).
1. Запишите образ Windows на флешку (например, через Rufus).
2. Возьмите файл **`autounattend.xml`** (находится прямо в корне этого репозитория) и просто скопируйте его в **корень вашей загрузочной флешки**.
3. Загрузитесь с флешки. 
> ⚠️ **БЕЗОПАСНОСТЬ ДАННЫХ:**
> Весь процесс установки ОС и создания профилей полностью автоматизирован. Однако **выбор и разметка жесткого диска намеренно оставлены ручными**, чтобы исключить случайное форматирование разделов с вашими личными файлами или резервными копиями.

### Способ 2. Умный онлайн-загрузчик (Для уже установленной ОС)

Если система уже установлена, вам не нужно скачивать весь репозиторий вручную!

* Просто скачайте и запустите один файл — **`loader.ps1`**.
* Это умный загрузчик, который работает даже на абсолютно чистой Windows. Он сам подтянет актуальные версии менеджера и всех модулей из репозитория, а затем автоматически запустит графический интерфейс.

## 🖥️ Менеджер автоматизации (GUI Диспетчер)

Главный среди всех процессов настройки. Это стильное приложение, которое берет на себя всю рутину, избавляя вас от необходимости смотреть в черные окна консолей.

**Ключевые особенности менеджера:**

* **Координация задач:** Строгий последовательный запуск модулей настройки один за другим.
* **Визуализация:** Понятный и красивый интерфейс в стиле Cyberpunk с анимированным прогресс-баром и статусом каждой задачи.
* **Отказоустойчивость:** Защита от сбоев — если при установке одной из программ произойдет ошибка (например, недоступен сервер), процесс не прервется и пойдет дальше.
* **Прозрачное логирование:** Незаметная фоновая запись всех действий скриптов в отдельный лог-файл для удобной проверки.

## ⚙️ Архитектура модулей (Этапы настройки)

После запуска диспетчер по очереди выполняет следующие скрипты из папки `Scripts`:

* 🧹 **`clean-and-photo.ps1` — Оптимизация ОС**
Глубокая очистка встроенного телеметрического и рекламного мусора (Debloat), возврат классического и быстрого средства просмотра фотографий, а также умная автоматическая настройка файла подкачки под объём вашей оперативной памяти.
* 🧩 **`install-sys-components.ps1` — Системные компоненты**
Тихая установка фундаментальных библиотек (Visual C++, DirectX, .NET), чтобы любые современные игры и тяжелый профессиональный софт работали без ошибок.
* 📦 **`apps-install.ps1` — Базовый пакет ПО**
Надежная установка повседневных программ по прямым ссылкам с автоматической проверкой целостности. Набор включает: *Google Chrome, Steam (отключается в автозагрузке для экономии ресурсов), WinRAR, qBittorrent, ShareX и K-Lite Codec Pack*.
* 📝 **`office-install.ps1` — Развертывание Microsoft Office**
Скрипт для полностью автоматического скачивания, установки и локальной активации стабильного пакета Microsoft Office 2024 Professional Plus LTSC. В целях оптимизации устанавливается только рабочий базис: Word, Excel и PowerPoint.
* 🔥 **`reset-setup-scripts.ps1` — Финальная зачистка (Режим камикадзе)**
Завершающий этап уборки. Скрипт очищает кэш инсталляторов и бесследно стирает всю папку `Scripts` вместе с собой с диска. Система остается в девственно чистом порядке перед финальной перезагрузкой.
